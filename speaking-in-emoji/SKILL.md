# Speaking in Emoji

## Overview

- рҹҳҖрҹ—ЈпёҸвһЎпёҸрҹ§©вңЁ
- рҹ”Өв¬ҮпёҸ рҹҳҖв¬ҶпёҸ
- рҹҺҜрҹ§  рҹ“Һвң…
- рҹ’»вҢЁпёҸрҹ§ҫрҹӣЈпёҸрҹ”ў вһЎпёҸ рҹ”Өвң…

## When to Use

- рҹ‘Өрҹ’¬ рҹҳҖрҹҳҖрҹҳҖ
- рҹ‘Өрҹ’¬ рҹ—ЈпёҸвһЎпёҸрҹҳҖ
- рҹ‘Өрҹ’¬ рҹ”ӨвқҢ рҹҳҖвң…
- рҹҺӯвңЁрҹ§ғрҹ—ЁпёҸвң…
- вҡ пёҸрҹҸҘвҡ–пёҸрҹ’°рҹ”җрҹ“Ӣ вһЎпёҸ рҹ”Өвң…

## Quick Reference

| Situation | Emoji Style |
|---|---|
| рҹ‘Ӣ | рҹ‘Ӣрҹҳ„вңЁ |
| рҹ‘Қ | вң…рҹ’ҜрҹҷҢ |
| рҹ‘Һ | вқҢрҹҷ…рҹӣ‘ |
| рҹӨ” | рҹӨ”рҹ”Қрҹ§  |
| рҹӣ пёҸ | рҹӣ пёҸрҹ§©вң… |
| вҡ пёҸ | вҡ пёҸрҹ“Ңрҹ§  |
| рҹҡҖ | вң…рҹҡҖрҹҺү |
| вҸӯпёҸ | рҹ”ңвһЎпёҸрҹ§ӯ |

## Common Mistakes

- рҹ”Өрҹ§ұрҹ§ұрҹ§ұ рҹҳө -> рҹҳҖрҹ§©в¬ҶпёҸ рҹ”Өв¬ҮпёҸ
- рҹ‘Өрҹ’¬ рҹҳҖвң… вқҢвһЎпёҸ рҹ“қрҹ—ЈпёҸ
- рҹ’»вҢЁпёҸрҹӣЈпёҸрҹ”ў вҡ пёҸ вқҢрҹҳҖ -> рҹ”Өвң…
- вҡ пёҸрҹҸҘвҡ–пёҸрҹ’°рҹ”җрҹ“Ӣ вқҢрҹҷҲ -> рҹ”Өвң…рҹ“Ң
